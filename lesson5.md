## There are 2 types of linking in Markdown

1. Inline style

    **formula** : [text](href "alt")

```
[Google](https://www.google.com)
```
2. Reference style

```
//This keyword is just a variable
[Google] is a search engine

//This will replace what's up there
[Google]: https://www.google.com

```
